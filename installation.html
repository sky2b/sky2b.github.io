Installation

Download an installation package for your operating system. For Linux and Mac it comes as a tar archive containing addgears executable and example projects. Windows installer adds application directory to the PATH variable making addgears available anywhere at the command prompt. It also creates a desktop shortcut (which opens the command prompt in projects directory) and adds shortcuts in the Start menu (including the one which opens the projects directory in Explorer).

After the installation is finished test it. At the command prompt type:

addgears -h

-config string
      makefile name (default "xmake.yaml")
-run string
      task to run (default "default")

As you can see, although, technically, AddGears is a command line application, there are only two options available. One is to specify a configuration file and another - task(s) to execute. The application actually is mainly controlled through configuration files. To work with these configuration files you may wish to install some editor, for example open source Atom. As always when working with textual files, it is advisable to use some version control system, for example git, optionally with some GUI client, for example, with free SourceTree for Windows/Mac . On Windows you may wish to install nicer console instead of standard command prompt, for example cmder with tabbed interface and better clipboard support. Also feel free to copy example configuration files in your working directory - as long as addgears is present in your PATH you may work in any directory you like.
