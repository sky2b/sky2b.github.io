Integrates easily

AddGears strives to deliver effective and affordable tools for business automation and data processing. To achieve this goal we follow the best practices and borrow from the established open source technologies.

From the ground up AddGears is designed as a widely open platform. It is built on top of popular node-webkit project, which allows to develop rich desktop applications using HTML5, CSS3, node.js and other open web standards and technologies.

The addgears package provides tools to integrate AddGears with node.js. It allows to execute node.js scripts in AddGears with full access to AddGears internals. Already mature and constantly growing node.js ecosystem makes it easy to extend AddGears functionality and enables AddGears users, when necessary, efficiently use any existing technology.

Intuitive tools

AddGears is developed in Javascript which makes it very natural to use Javascript to provide AddGears extension points. We think that Javascript and node.js ecosystem is a very productive, interactive environment, ideal for scripting, UI and visualization. AddGears integrates seamlessly with node.js and also provides tools to explore, process and chart data. It goes even further, for better scripting supporting a few Javascript extensions. This makes AddGears a good fit for data analyses and re-pricing.

AddGears is implemented on top of web technologies. You can extend AddGears and create a rich UI experience just leveraging your old HTML and CSS skills and resources. To help with this AddGears provides simple development tools.

Finally, as AddGears runs in a browser and has a full access to HTML DOM and other browser capabilities, it makes AddGears ideal for data harvesting and web scraping. To this end, the application provides intuitive visual tools to interactively parse web pages for data mining and competition analyses.

Customisation and Enterprise grade services

Every business is as unique as the people who are doing it. And ideally business needs IT tools which solve just exactly the problems the company is facing. For this reason we see AddGears first of all as a development platform, which should help cost-effectively develop customized solutions.

Our addgears node.js package provides tools to manage AddGears extensions. Another grunt-addgears package provides tools to turn ordinary HTML pages and node.js applications into AddGears extensions. Along with karma-addgears-launcher, integrating AddGears with Karma test runner, these tools turn AddGears into the comprehensive development platform.

Beside Community Edition, sponsored through the donations, we provide custom solutions development. For details contact our sales.
